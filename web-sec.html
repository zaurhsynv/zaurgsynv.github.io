Web Security Haqqında

Web Security, web tətbiqlərinin təhlükəsizliyini təmin etmək və potensial zəiflikləri aşkar etmək üçün istifadə olunan metodlar və texnologiyalar toplusudur.

Web Security sahəsində əsas fokus nöqtələri:

  • OWASP Top 10 zəifliklərinin analizi
  • Web tətbiqlərinin penetrasiya testləri
  • API təhlükəsizliyi
  • Cross-Site Scripting (XSS) müdafiəsi
  • SQL Injection qarşısının alınması
  • Web Application Firewall (WAF) konfiqurasiyası
