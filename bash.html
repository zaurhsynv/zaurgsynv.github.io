Bash Scripting

Linux Avtomatlaşdırmasının Gücü

1. Giriş: Bash nədir?

Bash (Bourne Again SHell) — Unix və Linux əsaslı əməliyyat sistemlərində işləyən ən geniş yayılmış komanda sətri interpreteridir. Bash yalnız interaktiv terminal mühiti üçün deyil, həm də sistem əməliyyatlarının avtomatlaşdırılması üçün ssenarilər yazmaq üçün geniş istifadə olunur.

2. Bash Scripting nədir?

Bash scripting, ardıcıl əmrləri .sh uzantılı fayl daxilində yazaraq onları avtomatik və sistemli şəkildə icra etməyə imkan verən bir texnikadır. Bu ssenarilər, sistem adminləri, DevOps mütəxəssisləri, və təhlükəsizlik mühəndisləri üçün əvəzolunmaz alətlərdir.

3. Bash Scripting ilə Nələr Etmək Olar?

  • Sistem ehtiyat nüsxələrinin avtomatlaşdırılması
  • Şəbəkə monitorinqi
  • Sistem log analizləri
  • Fayl sisteminin idarə olunması
  • Paketlərin avtomatik quraşdırılması
  • Web pentest avtomatlaşdırması
  • CTF challenge-lərdə tez exploit prototipləmə

4. Bash Skriptinin Quruluşu

Ən sadə skript belə görünür:

#!/bin/bash
echo "Salam Dünya"

Tərkib hissələri:

  • #!/bin/bash — shebang, skriptin Bash tərəfindən icra olunacağını göstərir
  • echo — ekrana mesaj çıxaran əmrdir

5. Əsas Strukturlar

5.1. Dəyişənlər

ad="Pərvanə"
echo "Salam, $ad!"

5.2. İf-Else Blokları

if [ -f /etc/passwd ]; then
  echo "Fayl mövcuddur"
else
  echo "Fayl tapılmadı"
fi

5.3. Dövrlər (Loops)

for dövrü:

for i in {1..5}
do
  echo "Nömrə: $i"
done

while dövrü:

x=1
while [ $x -le 5 ]
do
  echo "Dəyər: $x"
  ((x++))
done

6. Funksiyalar

salamla() {
  echo "Salam, $1"
}

salamla "Cyberus"

7. Fayl və Kataloq Əməliyyatları

Fayl mövcuddurmu?

if [ -e myfile.txt ]; then
  echo "Fayl var"
fi

Kataloq yaratmaq:

mkdir -p /var/backups/$(date +%F)

8. Bash-da Şəbəkə Əmrləri

  • Ping test: ping -c 1 8.8.8.8
  • Port yoxlaması: nc -zv 192.168.1.1 22
  • Web səhifə yükləmək: curl https://example.com

9. Kiber Təhlükəsizlikdə Bash Scripting

9.1. Recon (Kəşfiyyat)

Subdomain enumeration:

for sub in www mail ftp test
do
  host $sub.target.com
done

9.2. Port scanning (nc ilə)

for port in {20..25}
do
  nc -zv 192.168.1.100 $port 2>&1 | grep succeeded
done

9.3. Wordlist Brute-force

while read password; do
  echo "Yoxlanır: $password"
done < wordlist.txt

10. Real Dünya Ssenariləri

Ssenari 1: Günlük log backup skripti

#!/bin/bash
tar -czf /backup/logs_$(date +%F).tar.gz /var/log

Ssenari 2: SSH girişlərini yoxlayan skript

grep "Accepted password" /var/log/auth.log | awk '{print $1,$2,$3,$11}'

Ssenari 3: Web server status monitoru

status=$(curl -s -o /dev/null -w "%{http_code}" https://example.com)
if [ $status -ne 200 ]; then
  echo "Sayt DOWN: $status"
fi
