Red Teaming Haqqında

Red Teaming, təşkilatın təhlükəsizlik sistemlərini real dünya hücumlarına bənzər şəkildə sınaqdan keçirmək üçün istifadə olunan bir yanaşmadır. Bu proses, təşkilatın təhlükəsizlik müdafiəsini gücləndirmək və zəif nöqtələri aşkar etmək üçün həyata keçirilir.

Red Teaming əməliyyatları aşağıdakı sahələri əhatə edir:

  • Şəbəkə penetrasiya testləri
  • Fiziki təhlükəsizlik sınaqları
  • Social engineering hücumları
  • Web tətbiqlərinin təhlükəsizlik analizi
  • Active Directory təhlükəsizlik qiymətləndirməsi

Red Teaming Proyektləri

Enterprise Network Penetrasiya Testi

Böyük bir şirkət şəbəkəsində həyata keçirilən tam ölçülü penetrasiya testi. Active Directory, web tətbiqləri və şəbəkə infrastrukturu üzrə təhlükəsizlik qiymətləndirməsi.

Detallar

Social Engineering Kampaniyası

İşçilərin təhlükəsizlik mədəniyyətini qiymətləndirmək üçün həyata keçirilən phishing və vishing kampaniyası.

Detallar

Fiziki Təhlükəsizlik Testi
