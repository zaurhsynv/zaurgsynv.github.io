Active Directory Haqqında

Active Directory, Windows şəbəkələrində istifadəçi və resurs idarəetməsi üçün istifadə olunan bir dizin xidmətidir. Təhlükəsizlik baxımından Active Directory-nin düzgün konfiqurasiyası və müdafiəsi vacibdir.

Active Directory təhlükəsizliyi sahəsində əsas fokus nöqtələri:

  • Domain Controller təhlükəsizliyi
  • Group Policy təhlükəsizliyi
  • Kerberos protokolunun təhlükəsizliyi
  • Privileged Access Management (PAM)
  • Active Directory hücum və müdafiə texnikaları
